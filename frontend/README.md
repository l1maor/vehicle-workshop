# frontend

## Installation

Install the application dependencies by running:

```sh
npm install
# or
yarn install
# or
bun install
```

## Development

Start the application in development mode by running:

```sh
npm run dev
# or
yarn dev
# orun run dev
```

## Production

Build the application in production mode by running:

```sh
npm run build
# or
yarn build
# or
bun run build
```
